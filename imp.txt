Medical Center Login
    └── Full Access
        ├── Create/Delete Medical Centers
        ├── Create/Delete Regular Users
        ├── View All Users
        └── All Other Operations

Regular User Login
    └── Limited Access
        ├── View Own Profile
        ├── Update Own Profile
        ├── Create Blood Requests
        ├── View Donor/Patient Info
        └── Cannot Create Users/Centers

Request Comes In
    ├── Is User Authenticated?
    │   └── Yes
    │       ├── Is Medical Center?
    │       │   └── Yes: Full Access
    │       └── Is Regular User?
    │           └── Yes: Check Operation
    │               ├── Create/Delete Users? -> Deny
    │               ├── Create/Delete Centers? -> Deny
    │               └── Other Operations? -> Allow
    └── No: Deny Access
